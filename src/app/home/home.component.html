Angular Cheat Sheet

S.no Command description
1. npm install -g @angular/cli To install angular-cli to the system
2. ng new project-name To create a new angular application
3. ng serve It will serve the application on 4200 port
4. ng build It is used to build a project in the dist folder
5. ng doc It is used look for angular documentation
6. ng help It is used to get help on angular cli commands
7. ng generate component component-name (or)
ng g c component-name		 It is used to create component
8. ng generate directive directive-name (or)
ng g d directive-name		 To generate a directive
9. ng generate service service-name (or)
ng g s service-name		 To generate a service
10. ng generate class class-name (or)
ng g cl class-name		 To generate a class
11. ng generate interface interface-name (or)
ng g i interface-name		 To generate an interface
12. ng generate pipe pipe-name (or) ng g p pipe-name To generate a pipe
13. ng generate module module-name (or)
ng g m module-name		 To generate a module
14. ng g m module-name --routing To generate module with routing file
